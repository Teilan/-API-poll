# -API-poll

Запуск приложения  
Создайте окружение:  
```
python -m venv venv
```
Активируйте виртуальное окружение:

Windows:  
```
.\venv\Scripts\activate
```

macOS/Linux:  
```
source venv/bin/activate
```

Установите все зависимости:    
```
pip install -r requirements.txt
```

Выполните команду:
```
python main.py
```
